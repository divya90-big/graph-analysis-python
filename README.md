# Graph Analysis and Visualization using Python

This project demonstrates graph creation, graph analysis and graph visualization using Python.

## Overview

The program creates a graph structure by connecting multiple nodes through edges. It then converts the graph into an adjacency matrix representation and visualizes the graph using Python libraries.

## Features

* Creates graph structures using Python
* Connects nodes using edge relationships
* Generates adjacency matrix representation of the graph
* Visualizes network connections using NetworkX
* Displays graph structure for network analysis

## Libraries Used

* NetworkX
* Matplotlib

## Concepts Used

* Graph Theory
* Adjacency Matrix
* Network Visualization
* Node and Edge Representation

## Purpose

To understand graph theory concepts and visualize relationships between connected nodes using Python.
